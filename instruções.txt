## Iniciação em Windows:

1 - Ao descompactar a pasta, abra o cmd/terminal e execute os seguintes comandos:

pip install virtualenv
pip -m venv venv
cd venv\Scripts
activate
cd ..\..

## Iniciação Linux:
pip3 install virtualenv
pip3 -m venv venv
source myvenv/bin/activate

* Se der algum problema com o pip, normalmente avisa, mas talvez seja para instalar o pacote pip:
Windows: python get-pip.py
Linux: sudo apt install python3-pip 

** Mantenha o terminal aberto, pois ele estará com o ambiente virtual ativado

2 - Com o ambiente virtual ativado, instale os requisitos:
Windows: pip install -r requirements.txt
Linux: pip3 install -r requirements.txt

2 - Abra a pasta na IDE que utiliza e repita o processo de ativação do ambiente virtual

3 - No Mysql, crie o banco de dados para uso na aplicação:

CREATE DATABASE LivrariaIGE;

CREATE USER 'admin'@'localhost' IDENTIFIED BY 'admin';

GRANT ALL PRIVILEGES ON LivrariaIGE . * TO 'admin'@'localhost';
FLUSH PRIVILEGES;

4 - No terminal (IDE ou CMD), faça as migrações da base de dados:
Windows:
python manage.py makemigrations
python manage.py migrate

Linux:
python3 manage.py makemigrations
python3 manage.py migrate

* makemigrations: cria "scripts" para a criação dos dados no banco de dados
* migrate: roda os "scripts" e cria as tabelas no banco de dados

5 - Criação de super usuário para todas as permissões no sistema:
Windows: python manage.py createsuperuser
Linux: python3 manage.py createsuperuser

E preencha os campos com dados fáceis de lembrar. Não precisa preencher o email, apenas dê Enter se preferir não utilizar

6 - Para rodar o servidor, digite em algum dos terminais (preferencialmente o CMD, aberto primeiro):
Windows: python manage.py runserver
Linux: python3 manage.py runserver

* Estará rodando no caminho 127.0.0.1:8000, mas pode acessar também em localhost:8000

** Para encerrar, com o terminal aberto, aperte CTRL + C

7 - URLs:
/ : pagina home
/admin : sistema de administração, CRUD de todos os dados manualmente
/autor/<slug:slug> : abre o perfil de algum autor
/livros/ : lista os livros
/livros/<slug:slug> : dados sobre um livro especifico
/livros/adicionar/ : rota para adicionar livros (apenas acessível para usuários no grupo /"Professor")
/livros/<slug:slug>favoritar/ : favoritar um livro (ainda em desenvolvimento)
/livros/<slug:slug>/desfavoritar/ : desfavoritar um livro (ainda em desenvolvimento)
/registrar/ : registrar usuario
/login/ : realizar login
/logout/ : logout (não funciona manualmente)
/perfil/<username> : perfil de algum usuário

- Definição de slug:
Um slug, no contexto de um blog ou site, é uma parte da URL que identifica e descreve o conteúdo de uma página específica. Ele geralmente é criado a partir do título do artigo ou página, removendo-se símbolos, espaços em branco e caracteres especiais, e substituindo-os por hífens.
